About the Project
This project is a React + TypeScript web application built with modern frontend technologies. It provides a clean and interactive UI using Tailwind CSS and shadcn-ui components.

Key Features:
Responsive design
Fast development with Vite
Component-based architecture with React
Easy customization with Tailwind CSS

How to Run Locally
Follow these steps to run the project on your local machine:

Clone the repository
git clone <YOUR_GIT_URL>

Go to the project directory
cd <YOUR_PROJECT_DIRECTORY>

Install dependencies
npm install

Fix any vulnerabilities (optional)
npm audit fix

Start the development server
npm run dev

You can now open your browser and see the project running locally with live reloading.
Technologies Used
Vite – Fast development build tool
TypeScript – Strong typing for safer code
React – Component-based UI library
shadcn-ui – Prebuilt UI components
Tailwind CSS – Utility-first CSS framework
Deployment


To connect a custom domain, go to Project → Settings → Domains → Connect Domain.
